## Getting Started

### Installing

clone the repo and then install the dependencies e.g.

1. Clone the repo:

```bash
git clone git@github.com:flanderos/semesterproject1.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
